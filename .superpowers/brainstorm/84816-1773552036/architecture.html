Key Changes from Current

вң“ New: CLI Streaming

  • --stream flag for real-time output
  • Phase markers: [context] [review] [stream] [enrich]
  • Severity summary at end
  • Works directly in Claude Code Bash

вң“ New: TUI as Monitor

  • Watches session store for live updates
  • 3-panel layout: Live | Sessions | Findings
  • Auto-refreshes when CLI writes new session
  • No review execution in TUI (read-only)

вҶ» Changed: MCP Tools

  • MCP stays for programmatic access
  • But CLI becomes the recommended path
  • MCP prompt tells Claude to use CLI instead

вң— Removed from TUI

  • ReviewSetup (tool/mode selection UI)
  • ReviewProgress (progress bar)
  • All review execution logic
  • TUI becomes pure monitor/viewer
