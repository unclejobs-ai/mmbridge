Key Design Decisions vs Goyo Reference

Goyo에서 가져온 것

  • 상단 메타데이터 바 (PROJECT, SESSIONS, RUNNING, ELAPSED)
  • 테이블 형태 어댑터 상태 (NAME, STATUS, TIME 컬럼)
  • 하단 EVENTS 로그 (타임스탬프 + 이벤트 스트림)
  • 오른쪽 STREAM 출력 (실시간 텍스트 흐름)
  • 진행률 바 (████░░ 33%)

mmbridge에 맞게 변형한 것

  • Goyo의 밝은 테마 → Catppuccin Mocha 유지 (터미널 일관성)
  • Goyo의 에이전트 그래프 → 어댑터 테이블로 단순화
  • EVENTS 로그에 severity 하이라이트 추가
  • Sessions 탭 3컬럼 유지
